Para poder ver los recursos podemos iniciar la aplicación y poner en el navegador:

http://localhost:8080/v2/api-docs

Para lo cual necesitaremos algún pluggin de formateo de json.

Pero para evitar eso podemos directamente usar: 

http://localhost:8080/swagger-ui.html

Donde tendremos herramientas graficas para ver las definiciones.
